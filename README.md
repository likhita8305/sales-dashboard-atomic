# Sales Dashboard using Atomic Design

This project is a modern and responsive Sales Analytics Dashboard built using **Next.js** and **TypeScript**, following the **Atomic Design methodology**. The application presents sales data through a clean, structured user interface with reusable components and interactive visualizations.

The primary goal of this project is to demonstrate scalable frontend architecture, component reusability, and dashboard design suitable for real-world business applications.

## Project Architecture (Atomic Design)

The project follows Atomic Design principles to ensure scalability and reusability.

components/
├── atoms/ # Basic UI elements (buttons, text, icons)
├── molecules/ # Reusable component combinations (cards, inputs)
├── organisms/ # Complex UI sections (charts, tables)
├── layouts/ # Dashboard layout components
└── templates/ # Page-level layout structures

This architecture improves consistency, readability, and long-term maintainability.
